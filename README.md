# 🥁 Drum Kit

An interactive, browser-based virtual drum kit that lets you play drum sounds using your **keyboard keys** or by **clicking on-screen drum pads**. Built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

---

## 🎬 Demo

> Try it live: https://myself-vishwajeet.github.io/Drum_kit/

---

## ✨ Features

- 🎹 **Keyboard Support** — Press keys to instantly trigger drum sounds
- 🖱️ **Click to Play** — Click any drum pad with your mouse
- 💥 **Visual Feedback** — Drum pads animate on every hit for a real drumming feel
- 🔊 **Multiple Drum Sounds** — Each key maps to a unique drum sound (kick, snare, hi-hat, etc.)
- ⚡ **Zero Dependencies** — Pure vanilla JavaScript, no libraries required

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Structure & audio elements |
| CSS3 | Styling, animations & layout |
| JavaScript | Event listeners, sound playback & DOM manipulation |

## 🚀 Getting Started

### Prerequisites

All you need is a modern web browser. No installation required.

### Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/myself-vishwajeet/Drum_kit.git

# 2. Navigate into the project folder
cd Drum_kit

# 3. Open index.html in your browser
open index.html
```

Or simply double-click `index.html` to open it in your default browser.

## 🎮 How to Play

| Action | Result |
|--------|--------|
| Press `W` | Play Crash sound |
| Press `A` | Play Kick Bass Drum |
| Press `S` | Play Snare |
| Press `D` | Play Tom-1 |
| Press `J` | Play Tom-2 |
| Press `K` | Play Tom-3 |
| Press `L` | Play Tom-4 |

> 💡 You can also **click** directly on any drum pad to play the sound.
